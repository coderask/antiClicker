    Create src/renderer/src/map/MapView.tsx.

    Interface:
      interface PinCoords { latitude: number; longitude: number }
      interface MapViewProps {
        pin: PinCoords | null;
        onPinChange: (coords: PinCoords) => void;
      }

    Implementation:
    - useRef for the container div and the Map instance and Marker instance.
    - useEffect (runs once on mount):
      - Create new maplibregl.Map({ container, style: EOX_STYLE, center: [0, 0], zoom: 2, maxZoom: 14 })
      - on('click') handler: call onPinChange with the clicked lngLat
      - Cleanup: return () => map.remove()
    - useEffect (runs when pin changes):
      - If pin is null, remove marker if exists
      - If pin is set:
        - If no marker: create new maplibregl.Marker({ draggable: true }), add to map, listen to dragend
        - If marker exists: marker.setLngLat([pin.longitude, pin.latitude])
        - dragend handler: call onPinChange with marker.getLngLat()
    - Container div: style={{ width: '100%', height: '100%' }}, ref={containerRef}
    - Store onPinChange in a ref so the event listener always calls the latest version

    EOX_STYLE constant (top of file, outside component):

      sources: {

          type: 'raster',

          attribution: 'Sentinel-2 cloudless by EOX IT Services GmbH (2020)',
          maxzoom: 14,
        },
      },
