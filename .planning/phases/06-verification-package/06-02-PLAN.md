### Add verifySpoof IPC handler

In `src/main/ipc.ts`:
- Import `BrowserContext` type from playwright
- Add `VerifySpoofPayloadSchema = z.object({ id: z.string() })`
- Handle `IpcChannels.LauncherVerifySpoof`:
  - Get the launcher entry's context via registry
  - Get pages()[0] or open a new page
  - Evaluate `navigator.geolocation.getCurrentPosition` returning coords
  - Get expected coords from registry
  - Return `{ reported: {lat, lng}, expected: {lat, lng}, match: boolean }`
  - Match: `Math.abs(reported.lat - expected.lat) < 0.001 && Math.abs(reported.lng - expected.lng) < 0.001`

Implementation note: use `page.evaluate()` with a serializable function string, not a function reference (preload boundary restriction doesn't apply here — this is main process calling Playwright directly).
</task>
